# 知識管理助理使用說明

## 快速流程：上傳文件 → 勾選 → 提問

---

## 1. 文件管理（左欄）

### 文件狀態

| 狀態 | 說明 |
|------|------|
| 處理中 | 上傳後系統正在進行切塊與向量化，請稍候 |
| 可用 | 向量化完成，可參與 RAG 檢索 |
| 失敗 | 處理失敗，可刪除後重新上傳 |

### 上傳文件

- 支援格式：**PDF、純文字（.txt）、Markdown（.md）**。
- 點擊「上傳文件」按鈕或拖曳檔案至上傳區。
- **個人 / 公共**（管理員可見）：
  - **個人**：僅自己可存取，上傳後預設勾選。
  - **公共**：所有成員可存取，需管理員（admin / manager）才能上傳或刪除。

### 勾選文件

- 只有**已勾選且狀態為「可用」**的文件才會納入本次對話的 RAG 檢索範圍。
- 可使用頂端的全選 checkbox 一次勾選 / 取消全部。
- 右側對話輸入框上方會顯示「已勾選 N / 可用 M 份」。

---

## 2. 對話區（右側）

- 在輸入框輸入問題，按 Enter 或送出按鈕發送。
- AI 會從**勾選文件**中進行語意檢索，找出相關段落後回答。
- **清除對話**：點對話區右上角「清除」圖示，清空目前的對話紀錄。

---

## 3. AI 設定區（右側面板）

- 可調整 **AI 模型、回應語言、回應風格（詳細 / 精簡）、角色** 等參數。
- 設定僅對目前工作階段有效，不影響其他使用者。
- 詳細說明請參閱「AI 設定」線上說明。

---

## 注意事項

- 未勾選任何文件時，送出問題會提示先勾選文件。
- 刪除公共文件需管理員以上權限；個人文件僅本人可刪除。
- 大型文件（> 10 MB）向量化時間較長，請等候狀態變為「可用」再提問。
